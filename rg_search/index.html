ripgrep (rg)

ripgrep is a line-oriented search tool that recursively searches the current directory for a regex pattern. By default, ripgrep respects gitignore rules and automatically skips hidden files, binary files, and symlinks. ripgrep has first class support on Windows, macOS, and Linux, with binary downloads available for every release.

Installation

# macOS — Homebrew
brew install ripgrep

# Windows — winget
winget install BurntSushi.ripgrep.MSVC

# Arch Linux
pacman -S ripgrep

Quick Examples

# Search for a pattern in all files
rg "fn main" ./src

# Only show files that contain the pattern
rg -l "TODO\|FIXME"

# Search with file type filtering
rg --type rust "impl Debug"

# Count matches
rg -c "error" /var/log/app.log

Features

  • Respects .gitignore by default
  • Automatic filtering of binary files
  • Regex engine backed by Rust's regex crate
  • Supports PCRE2 for look-around and backreferences
  • Replace mode for find-and-replace
  • JSON output for programmatic use
  • Parallel directory traversal
  • Unicode support including case-folding

Search Flags

ripgrep supports over 100 flags for customizing search behavior. Below are some commonly used options:

FlagDescriptionDefault
-i, --ignore-caseCase insensitive searchfalse
-t, --typeOnly search files matching TYPEnone
--max-depthDescend at most NUM directoriesnone
