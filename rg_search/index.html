A line-oriented search tool that recursively searches the current directory for a regex pattern. Respects gitignore rules and skips hidden/binary files by default. First-class support on Windows, macOS, and Linux.

README

Installation

# macOS — Homebrew
brew install ripgrep

# Windows — winget
winget install BurntSushi.ripgrep.MSVC

# From source
cargo install ripgrep

Quick Start

# Search for a pattern in all files
rg "fn main" ./src

# Only list matching files
rg -l "TODO"

# Count matches per file
rg -c "error" /var/log/

Key Features

Respects .gitignore by default. Automatic filtering of binary files. Regex engine backed by Rust's regex crate. Supports PCRE2 for look-around and backreferences. Replace mode for find-and-replace operations. JSON output for programmatic use. Parallel directory traversal for speed.
