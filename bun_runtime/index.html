⚡ Fast Runtime

Drop-in Node.js replacement. Starts 4x faster than Node.js with built-in TypeScript and JSX support.

📦 Bundler

Drop-in replacement for webpack, esbuild, and tsup. Bundle frontend apps with first-class CSS and SVG imports.

Jest-compatible test runner built in. No extra dependencies, no configuration needed.
