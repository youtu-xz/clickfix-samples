Installation

Shell
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

Configuration

~/.zshrc
# Enable the theme
ZSH_THEME="powerlevel10k/powerlevel10k"

# Configure prompt elements
POWERLEVEL9K_MODE="nerdfont-complete"
POWERLEVEL9K_LEFT_PROMPT_ELEMENTS=(dir vcs status)
POWERLEVEL9K_RIGHT_PROMPT_ELEMENTS=(time battery)

Features

⚡ Instant Prompt

Prompt appears instantly before Zsh finishes loading plugins. No more waiting to start typing.

👀 Git Status

Shows branch name, dirty state, ahead/behind commits, stash count, and rebase status in real time.

🎨 Customizable Segments

50+ prompt segments — directory, node version, k8s context, AWS profile, battery, time, and more.

🛠 Configuration Wizard

Run p10k configure for an interactive setup that guides you through every option.

View the complete p10k configuration reference with all 50+ segments, color schemes, and display styles.
